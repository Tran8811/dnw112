###Kiểm tra autoscaling thực tế trên môi trường GKE, JMeter
<img width="632" height="446" alt="image" src="https://github.com/user-attachments/assets/8b5713b2-01b6-4e65-be75-156a14aaa6e3" />
Các đường thể hiện:
Limit (màu tím): Giới hạn CPU có thể dùng tối đa của pod.
Requested (màu xanh lá): Lượng CPU được yêu cầu mặc định khi pod khởi tạo.
Used (màu xanh dương): Lượng CPU thực tế sử dụng.
